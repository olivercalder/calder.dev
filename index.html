Welcome to calder.dev

This website belongs to Oliver Calder. Hello! Thank you for visiting.

I am a student at Carleton College, studying mathematics and computer science. My academic interests also include linguistics, and I really enjoy the intersection between these three areas, in particular the encoding, modification, and expression of information through various means, from natural language to cryptography.

I am also an avid Linux user and a proponent of open-source software and first-principles problem solving. My language of choice is C, but I work frequently with Python and bash, I have worked extensively in the past with Rust, and I really enjoy using functional languages whenever the opportunity arises.

I am passionate about classical music. I am a clarinetist and pianist and an alumn of the Greater Twin Cities Youth Symphonies (GTCYS).

Research

  • Sonic Signatures: A machine learning and data visualization toolkit written to analyze phonological patterns in the speech of characters from Shakespeare's plays. Source code and a more verbose description can be found here. Advised by Eric Alexander (Department of Computer Science, Carleton College).
  • Character Text Pipeline: (still working on the name.) This is an extension of the work from the Sonic Signatures project, built to process texts from the EEBO-TCP project. This allows researchers to extract and clean the speech of individual characters from large xml files, and organize these characters' speech into a format which can be easily translated and analyzed, such as by the scripts from the Sonic Signatures project. Source code and a more verbose (and up-to-date) description can be found here. Advised by Eric Alexander (Department of Computer Science, Carleton College).
  • Exploratory Operating Systems: Upcoming, Summer 2020. Advised by Aaron Bauer (Department of Computer Science, Carleton College).
